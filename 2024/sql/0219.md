### 데이터 모델
데이터를 사용하려는 목적에 맞게 정리하고 체계화해놓은 모형


### 데이터 모델링

entity(개체)
- 저장하고 싶은 데이터의 대상

attribute(속성)
entity에 대해서 저장하려는 내용


relationship (관계)
entity들 사이 연결점

constraint(제약 조건)

relation(테이블)

relational model
entity relationship model (ERM)
- 개체와 관계로 모델링

데이터 모델 스펙트럼
- 개념모델(conceptual model)
- 논리모델(logical model)
- 물리모델(physical model)
